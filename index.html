TAGS DESCONTINUADAS NO HTML5.

O que são tags descontinuadas?

São tags que não são mais utilizadas e/ou foram substituidas por outras. Veja exemplos abaixo

< applet >
Insere um applet na página
Substuida por: <object>

< basefont >
Configura os estilos de fonte
Substuida por:Estilos de fonte CSS

< center >
Centraliza os elementos internos à tag
Substuida por: Estilo CSS. Ex:” text-align:Center”

< dir >
Lista de diretórios
Substuida por: <ul>

< u >
Underline
Substuida por: Estilos de fonte CSS

< strike >
Tacha o texto envolto
Substuida por: Estilos de fonte CSS

< font >
Configura estilo de fonte para o texto envolto
Substuida por: Estilos de fonte CSS

< isindex >
Adiciona campo de pesquisa
Substuida por: <form>

< menu >
Lista de menus
Substuida por: < ul >
